# RelayClip

RelayClip is a Tauri 2 desktop app for macOS and Windows that relays clipboard content across devices on the same LAN. It auto-discovers peers over mDNS, auto-trusts private-network devices with a TOFU model, and syncs text plus PNG clipboard payloads over `TCP + rustls + length-prefixed protobuf`.

## What is implemented

- Tauri 2 desktop app with a tray-first workflow and a Svelte 5 settings window
- LAN discovery via `_relayclip._tcp.local`
- Automatic trust for private-network peers and active-device switching
- Bidirectional clipboard sync for text and images
- Bilingual interface with English and Simplified Chinese for the window and tray menu
- JSON-backed device store under the local app data directory
- TLS transport with self-signed per-device identity and fingerprint checking

## Core layout

- `src/App.svelte`: macOS-inspired settings UI for devices, routing, and app preferences
- `src/lib/api.ts`: frontend bridge for Tauri commands and runtime events
- `src-tauri/src/runtime.rs`: application state, device trust, sync orchestration, and commands
- `src-tauri/src/discovery.rs`: mDNS announce and browse loop
- `src-tauri/src/transport.rs`: rustls server/client transport and protobuf envelope framing
- `src-tauri/src/clipboard.rs`: clipboard polling, PNG conversion, loop suppression
- `src-tauri/src/store.rs`: persistent state load/save
- `src-tauri/src/tray.rs`: dynamic tray menu and window restore behavior

## Commands

```powershell
pnpm install
pnpm check
pnpm build
pnpm tauri dev
pnpm tauri build --debug
```

Do not treat `src-tauri/target/debug/relayclip.exe` generated by a plain `cargo build` as the final desktop app binary. For local development use `pnpm tauri dev`; for a standalone Windows app use the MSI produced by `pnpm tauri build --debug -b msi`.

If you need to run Rust builds from a plain PowerShell session on Windows, start a Visual Studio developer environment first:

```powershell
cmd /c '"C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\Common7\Tools\VsDevCmd.bat" -no_logo && set PATH=%USERPROFILE%\.cargo\bin;%APPDATA%\npm;C:\Program Files\nodejs;%PATH% && pnpm tauri build --debug'
```
